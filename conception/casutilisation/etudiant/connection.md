
# Connection 

A la connection l'etudiant est connecté à son la page **base_page** 
avec pour contenu son **tableau_de_Bord**.

Le tableau de bord permet d'accèder aux élements suivants :

- **calendrier**
- **todoliste**
- **email**
- **tableaudescours**
- **rankings**
- **badges**
- **points** (ask)
- **externes** manara et autres
